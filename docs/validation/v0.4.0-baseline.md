### Validation Notes
- The Europe PMC connector currently accepts a sort parameter but does not apply native API-level chronological sorting.
- The internal `_map_epmc_sort()` implementation returns `None` for all sort modes.
- For All Sources retrieval, Europe PMC candidates are sorted centrally after aggregation.
- Validation confirmed that centralized sorting produces correct Most Recent and Oldest First ordering within the All Sources workflow.
