LitSearch v0.4.0 Release Notes

Release Date: June 2026

Overview

LitSearch v0.4.0 introduces a major architectural milestone with the completion and validation of the All Sources retrieval workflow.

This release focuses on retrieval consistency, cross-source aggregation, export reliability, and documentation improvements.

Cross-Source Deduplication

Implemented deterministic duplicate detection using:

Primary:

DOI matching

Fallback:

normalized title matching

This reduces duplicate records across integrated literature sources.

Relevance
