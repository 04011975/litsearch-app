# LitSearch v0.6.0 Roadmap

## Goal

Expand LitSearch by adding new scholarly retrieval sources, improving metadata quality, extending search capabilities, and enhancing export functionality while preserving the stable architecture introduced in v0.5.0.

The architectural principles established in v0.5.0 remain unchanged:

- centralized retrieval pipeline
- uniform Paper model
- centralized deduplication
- identical UI and export retrieval
- connector-independent architecture
